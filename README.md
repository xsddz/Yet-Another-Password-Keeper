
# Yet-Another-Password-Keeper

```
# 克隆这仓库
$ git clone https://github.com/xsddz/Yet-Another-Password-Keeper.git
# 进入仓库
$ cd Yet-Another-Password-Keeper
# 安装依赖库
$ npm install
# 运行应用
$ npm start
```
